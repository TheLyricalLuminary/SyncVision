SyncVision is a deterministic music rights verification and sync matching system. Every output must be identical across repeated runs given the same input. The tech stack is Node with Express and TypeScript for the backend, Prisma with Postgres for the database, Redis Streams for the job queue, and a Python worker using librosa for audio analysis. The frontend is React with Vite and TypeScript. Session One goal: build and migrate the Prisma schema only. No API routes, no business logic, no queue. Schema only.
